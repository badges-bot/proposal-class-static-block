A class static block environment is a Lexical Environment that corresponds to the evaluation of the static block of a class. A class static block environment establishes a new `this` binding.

Class Static Block Environment Records

A class static block Environment Record is a declarative Environement Record that is used to represent the top-level scope of a class static block and provides a `this` binding.

Class Static Block Environment Records have the additional state fields listed in .

Field Name Value Meaning
[[ThisValue]] Any This is the *this* value used for this invocation of the function.
[[ThisBindingStatus]] `"initialized"` | `"uninitialized"`
[[FunctionObject]] Object The function object for a class whose static block evaluation caused this Environment Record to be created.
[[HomeObject]] Object | *undefined* If the associated function has `super` property accesses and is not an |ArrowFunction|, [[HomeObject]] is the object that the function is bound to as a method. The default value for [[HomeObject]] is *undefined*.
